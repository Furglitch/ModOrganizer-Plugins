Troubleshooting

  • A plugin installed through Plugin Finder does not work.
  • A plugin I installed manually isn't listed as the correct version.
  • A plugin listed in Plugin Finder is out of date.
  • Updating or uninstalling a specific plugin always fails.
  • My plugin isn't listed in Plugin Finder.

A plugin installed through Plugin Finder does not work.

Plugins listed on Plugin Finder are manintained by their separate authors. If you have issues with a plugin, please check the documentation, Github and/or Nexus pages for the plugin.

A plugin listed in Plugin Finder is out of date.

Plugins listed on Plugin Finder are manintained by their separate authors. Please contact the author of the plugin and request they update their data for Plugin Finder.

A plugin I installed manually isn't listed as the correct version.

Plugin Finder attempts to detect the version based on the contents of the plugin, but this may not always be successful. When this fails, a default version number is given to the plugin. To remedy this, install or update the plugin through Plugin Finder.

Updating or uninstalling a specific plugin always fails.

Even if a message is logged in Mod Organizer that a file could not be moved/deleted, Plugin Finder will still attempt to move/delete those files when Mod Organizer restarts.

Sometimes current plugin files are locked or marked as read-only, to resolve this issue you will need to manually delete the current plugin from Mod Organizer's plugins folder.

My plugin isn't listed in Plugin Finder.

If you have a plugin that you would like to add to Plugin Finder, you will need to create a manifest file and host it on the Github for your plugin.

A generator to help generate a manifest can be accessed here:
https://kezyma.github.io/?p=pluginfinder-generator

Once your manifest is on Github, you will need to open a pull request to add your plugin to the directory here:
https://github.com/Kezyma/ModOrganizer-Plugins/blob/main/directory/plugin_directory.json

When your pull request is approved, your plugin will appear in Plugin Finder.
