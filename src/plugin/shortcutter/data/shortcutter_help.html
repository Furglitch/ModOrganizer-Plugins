Where is my shortcut?

All generated shortcuts are placed on the desktop, or where Windows believes your desktop is located.

Openinig Mod Organizer defaults to the last instance that Shortcutter opened.

The current Mod Organizer instance is set in the Windows registry, when Shortcutter launches from a shortcut, the instance in the registry is changed. Unfortunately there is no way to change it back afterwards, so this has to be done manually.
